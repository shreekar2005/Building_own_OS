Detailed Description

Manages multitasking and scheduling of tasks.

Implements a simple Round-Robin scheduler to switch between registered tasks.

◆ onTaskExit()

void TaskManager::onTaskExit ( )
static

A static helper function used as the return address for tasks.

When a task function returns (finishes), the CPU pops this address from the stack. This function effectively calls killCurrentTask() and halts the CPU until the next scheduler tick.

◆ schedule()

CPUState * TaskManager::schedule ( CPUState *  cpustate)

The core scheduling function called by the timer interrupt.

Saves the current task's state (passed in cpustate) and selects the next task to run. Uses a Round-Robin algorithm with a time quantum.

Parameters
cpustateThe register state of the task that was just interrupted.
Returns
The register state of the next task to run (which the interrupt handler will restore).
