Detailed Description

Represents a single unit of execution (a thread or process) in the OS.

Each Task maintains its own private stack memory (4KB) and a pointer to its saved CPU state.

◆ reset()

void Task::reset ( )

Resets or initializes the task's stack frame.

This function sets up a "fake" interrupt stack frame at the top of the stack array. It sets the instruction pointer (EIP) to the entrypoint and the stack pointer (ESP) to the onTaskExit handler. This must be called before the task is run for the first time.
