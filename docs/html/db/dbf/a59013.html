Detailed Description

Manages multitasking and scheduling of threads.

Implements a simple Round-Robin scheduler to switch between registered threads.

void KThreadManager::onThreadExit ( )
static

A static helper function used as the return address for threads.

When a thread function returns (finishes), the CPU pops this address from the stack. This function effectively calls killCurrentThread() and halts the CPU until the next scheduler tick.

CPUState * KThreadManager::scheduleThreads ( CPUState *  cpustate,
bool  forceSwitch = false 
)

The core scheduling function called by the timer interrupt.

Saves the current thread's state (passed in cpustate) and selects the next thread to run. Uses a Round-Robin algorithm with a time quantum.

Parameters
cpustateThe register state of the thread that was just interrupted.
forceSwitchif true then just schedule thread without looking at time quantum
Returns
The register state of the next thread to run (which the interrupt handler will restore).
