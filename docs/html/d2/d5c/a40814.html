◆ essential::CPUState

struct essential::CPUState

Represents the state of the CPU registers.

This structure matches the layout of registers pushed onto the stack by the interrupt handler (common/stub) before the C++ handler is called. When switching tasks, we save the old task's state here and load the new task's state from here.
